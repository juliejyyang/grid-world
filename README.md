simple implementation of policy iteration using DP on a gridworld MDP with the purpose of learning the fundamentals of RL

 ### Problem
 an agent navigates an NxN grid from the top-left corner (0,0) to the bottom right

 ### References
 based on policy iteration formulation in Sutton and Barto, *Reinforcement Learning: An Introduction*, Chapter 4. JavaScript reference implementation: Andrej Karparthy's [REINFORCEjs](https://cs.stanford.edu/people/karpathy/reinforcejs/gridworld_dp.html).
